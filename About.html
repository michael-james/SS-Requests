Understanding the Queue

Current

#

The request's current rank in the queue.

Status

Status Description
Received A new request.
Waiting for Start A request for which the files are not yet available.
Needs Information A request that cannot start until more information is gathered.
Reviewed A member of our Assistants team has reviewed the requests and its files and determined it is ready.
Assigned A request that is assigned but not yet started.
In-progress A request that has been started but is not ready for review.
Unresolved Issues A request that has one or more major issues that someone else needs to resolve before final delivery.
Pending Confirmation A request that the assistant has determined is ready for delivery but needs TC review and confirmation.
On-hold A request that has one or more issues that someone outside the Assistants team is currently working to resolve that is blocking final delivery.
Completed A request that the team and the TC have confirmed is complete and ready for delivery.
Cancelled A request that was cancelled and taken out of the queue.

Assignee

The assistant assigned to perform a request. Once someone is assigned to your request, they are the main point of contact so please direct any updates or questions to them.

Dates

Estimated Return Date [Est. Rt.]

The date, based on the current queue, on which the team estimates the request will be returned by the end of the work day of the TC who made the request. This date is just an estimate and is subject to change as new requests come in or requests take shorter or longer to complete than expected.
